Prueba Desarrollador Globaltek

GLOBALTEK DEVELOPMENT es una compañía que ofrece el servicio de Outsourcing de Tecnología de la Información, principalmente para Recursos Humanos Especializados, Soluciones de Desarrollo y Facilidades para clientes locales y globales.

Adicionalmente cuenta con una Fábrica de Software que ofrece servicios de Desarrollo de Software a la Medida, Productos de Caja, Soluciones para dispositivos móviles, Soluciones SaaS (Software como Servicio), Soluciones de Automatización, Consultoría y Suministro de Infraestructura.

Nació con el apoyo de tres diferentes empresas: Globaltek Security S.A. (GTS), MVA y Vipware Ltda. que aunaron esfuerzos para construir una Compañía que ofrece y entrega diferentes tipos de productos y servicios a sus clientes.
